# CHANGELOG.md

## [v1.0.0] - 2026-06-24

### Added
- **Первая стабильная версия проекта**
- Процедурная генерация ствола с текстурой коры (4096×4096)
- Рекурсивная система веток с реалистичным изгибом
- Генерация хвои через InstancedMesh (> 100 000 иголок)
- Система снега на ветках (повторяет форму ветвей)
- Динамическое освещение от солнца с управлением временем
- Декоративный циферблат в стиле ночного неба
- LOD-система (3 уровня детализации)
- Управление камерой (OrbitControls)
- Панель настроек (ветер, снег, температура)
- Синхронизация времени с реальным (кнопка "Сейчас")
- Отображение FPS в реальном времени
- Адаптивная вёрстка

### Technical
- Использование Three.js 0.128.0
- Модульная архитектура (ES Modules)
- Canvas 2D для генерации текстур
- Оптимизированные текстуры (сжатие через Canvas)
- Эффекты свечения солнца (Additive Blending)
- Автоматическое управление pixel ratio

### Performance
- InstancedMesh для хвои (оптимизация памяти)
- LOD-система для снижения нагрузки при отдалении
- Ограничение pixel ratio на слабых устройствах
- Динамическое отключение теней при низком FPS
